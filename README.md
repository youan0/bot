# 1337 Bot

## Requirements

- **Python3** installed
- and **selenium** & **pyttsx3** packages installed

```
pip install selenium
pip install pyttsx3
```

## How to use

### add your email and password to **_config.json_**

```
{
  ...
  "EMAIL": "example@gmail.com",
  "PASSWORD": "1234567",
  ...
}
```

### Then run the **_main.py_** script

```
python main.py
```
